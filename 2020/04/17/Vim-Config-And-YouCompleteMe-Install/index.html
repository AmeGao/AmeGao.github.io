Vim 配置和YouCompleteMe插件安装

一、Vim安装和配置

简介

Vim是从 vi 发展出来的一个文本编辑器。代码补全、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用，和Emacs并列成为类Unix系统用户最喜欢的文本编辑器。

vim的设计理念是命令的组合。用户学习了各种各样的文本间移动/跳转的命令和其他的普通模式的编辑命令，并且能够灵活组合使用的话，能够比那些没有模式的编辑器更加高效的进行文本编辑。同时vim与很多快捷键设置和正则表达式类似,可以辅助记忆。并且vim针对程序员做了优化。来源百度百科

Vim编辑器相关的所有功能开关都可以通过.vimrc文件进行设置。

上图中最下面的几行是vim读取配置文件的顺序，首先是安装目录内的vimrc文件，其次是用户主目录中的. vimrc 文件。安装目录内的vimrc文件是Vim编辑器的全局配置，对所有的用户都生效，而用户主界面的.vimrc文件是用户个人的配置，仅对该用户生效。这次配置vim就是配置单个用户的.vimrc文件。
